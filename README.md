## COMO INSTALAR ESTE BOT NO TERMUX
```bash
> pkg install nodejs && pkg install git
> git clone https://Venom-ofc/Venombot-tec
> cd Venombot-tec
> bash venom-pt.sh
> node venombot-pt
```
## COMO INSTALAR ESTE BOT NO PC
```bash
> git clone https://Venom-ofc/Venombot-tec
> cd Venombot-tec
> npm i
> node venombot-pt
```
